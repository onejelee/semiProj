css 파일을 모아놓는 폴더 입니다.

여기에 자신이 쓰는 css 파일을 넣어놓을 폴더를 만들어 주세요

폴더명은 다른사람이 알아보기 쉽도록 만드시면 됩니다. 

그 안의 파일명은 폴더명+'_' 을 앞에다 붙이고 파일이름을 써주시면 됩니다. 

ex ) 관리자 메뉴를 작업하는 사람의 폴더명 :  Admin 
폴더의 파일명 
Admin_XXXX_1.css
Admin_XXXX_2.css
Admin_XXXX_3.css
...
...
...

부트스트랩을 위한 bootstrap.css파일은 lib폴더를 만들어서 넣어주세요
